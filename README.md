# ☁️ DataStock

### *A Modern Cloud Storage Platform Inspired by Google Drive*

<img src="https://img.shields.io/badge/Status-Active-success?style=for-the-badge"/>
<img src="https://img.shields.io/badge/Frontend-React%20%2B%20Vite-blue?style=for-the-badge"/>
<img src="https://img.shields.io/badge/Backend-Node.js-green?style=for-the-badge"/>

</div>

**Overview**
DataStock is a full-stack cloud storage platform built to deliver a seamless file management experience inspired by Google Drive — combining a modern UI/UX with a scalable backend, secure authentication, cloud storage, and efficient file organization. Users can organize, preview, search, star, and share files through an intuitive dashboard.

### 🏗 Architecture

```text
              USER
               │
               ▼
        React Frontend
               │
     Axios REST API Calls
               │
               ▼
      Express.js Backend
               │
     Authentication Layer
               │
       Prisma ORM Engine
               │
          PostgreSQL DB
               │
          Cloudinary CDN
               │
        Secure File URLs
```

### 🛠 Tech Stack

| Category | Technology |
|---|---|
| Frontend | React + Vite |
| Styling | Tailwind CSS |
| Backend | Node.js + Express |
| Database | PostgreSQL |
| ORM | Prisma |
| Authentication | JWT + Google OAuth |
| Storage | Cloudinary |
| Deployment | Vercel + Render |

### 🎯 Why I Built This

Most storage apps only handle uploads. I wanted to understand how production-level applications actually work — so I combined authentication, cloud storage, relational database design, folder hierarchy, search, and a responsive UI into a single app. The goal wasn't just another repo on GitHub; it was learning how real-world software is designed end to end.
