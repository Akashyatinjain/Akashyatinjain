## 📖 Overview

DataStock is a full-stack cloud storage platform designed to deliver a seamless file management experience inspired by Google Drive.

The project focuses on combining modern UI/UX with scalable backend architecture, secure authentication, cloud storage, and efficient file organization.

Instead of simply uploading files, users can organize, preview, search, star, and share them through an intuitive dashboard.

# 🎯 Why I Built This

Most storage applications focus only on uploading files.

I wanted to understand how production-level applications work internally.

So I challenged myself to build a platform that combines:

- Authentication
- Cloud Storage
- Database Relationships
- Folder Hierarchy
- Search
- Responsive UI
- REST APIs
- Secure Backend

into a single application.

The goal wasn't just building another project—

It was learning how real-world software is designed.
